# Fudan_272cities_compound_heatwave
The R codes presented here support the results discussed in the manuscript currently under review by Nature Climate Change.

These codes have been developed to replicate the primary analyses conducted in the aforementioned study. The analysis is broadly divided into two stages: (1) Historical analysis using time-series methods, and (2) Quantification of impacts under the three Shared Socioeconomic Pathways (SSP) scenarios.

Please note that the provided codes are currently not fully annotated. We intend to provide a more comprehensive, standardized, and readable version of the code during the pre-acceptance phase.
